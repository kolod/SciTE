Файлы сокращений для языков программирования

Напечатайте сокращение и нажмите Ctrl+B (или Ctrl+Shift+R для выбора из списка), чтобы заменить его на полный текст

Другой вариант:
Выделите текст. Нажмите Ctrl+Shift+R для выбора из списка. Вставленное значение обрамит выделенный текст слева и справа.
Данный вариант удобно использовать для HTML, обрамляя нужными тегами выделенный текст.

В оригинальном SciTE используется единственный файл сокращений.
Подключение файла сокращений только для данного языка доступно только в SciTE - Ru <http://scite-ru.org/>

Файлы *.abbrev cодержат список сокращений в форме сокращение=полный текст
Символ "|" определяет положение курсора после вставки текста
или (если вставка производилась выбором из списка по Ctrl+Shift+R) то положение исходного текста

\n - перевод строки
\t - знак табуляции

Примеры:
main=int main(int argc, char *argv[]) {\n|\n}
i=if (|) {\n\n}
<ta=<table border="0" cellpadding="2" cellspacing="0" width="100%">\n\t<tr>\n\t\t<td>|</td>\n\t</tr>\n</table>

Примечание:
Файлы .abbrev удобно использовать для быстрой вставки в редактируемый файл многострочных конструкций кода часто употребляемых процедур.
(См. пример vbs.abbrev)
